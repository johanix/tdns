## tdns-cli auth zone list

List configured zones

```
tdns-cli auth zone list [flags]
```

### Options

```
  -f, --file      Show zone input file
  -h, --help      help for list
  -N, --notify    Show zone downstream notify addresses
  -P, --primary   Show zone primary nameserver
```
