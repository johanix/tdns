## tdns-cli auth keystore sig0 import

Add a new SIG(0) key pair to the keystore

```
tdns-cli auth keystore sig0 import [flags]
```

### Options

```
  -f, --file string   Name of file containing either pub or priv SIG(0) data
  -h, --help          help for import
  -z, --zone string   Zone to import SIG(0) key for
```
