## tdns-cli auth catalog group add

Add a group to the catalog's group list

```
tdns-cli auth catalog group add --cat <catalog-zone> --group <group-name> [flags]
```

### Options

```
      --cat string     Catalog zone name (required)
      --group string   Group name (required)
  -h, --help           help for add
```
